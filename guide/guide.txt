$text 2. Structure of a Wave script.

!size 20
!color fg

$nl

$text A Wave script is written in a plain text file. Wave scripts do not follow any extension. You can have an extention for a Wave script or not as long as it's a text file. It's better to have a <i>.txt</i> extension.

$nl

!color green

$text 2.1 Compiling a Wave script.

$nl

!color fg

$text A Wave script can be compiled by passing it to the Wave traspiler.

$nl

!color blue
!font Consolas

$text %sp%sp%sp%sp $ wave guide.txt

!color fg
!font Arial

!color fg

$nl
$text The transpiler generates a <i>.html</i> file of the same name. This is the final document.

$nl 2

$text A Wave script consists of three different types of keyword:

$nl

!points-type ordered
!points-style lower-roman
!color brown

$points The Page Property Keywords ; The Content Property Keywords ; The Content Keywords

$nl 2

!color green

$text 2.2 The Page Property Keywords

!color fg

$nl
$text The page property keywords start with a <i>~</i>. These properties apply to the entire document. These properties include some important elements that define the look of the document.

$nl

$text Here's a table of the page properties and their uses:

$nl

!align center
!color brown

$table <b>Page Property</b> ; <b>Use</b> ; <b>Example</b> | ~title ; Sets the title to tab in Browser Window ; <i>~title Hello!</i> | ~theme ; Sets theme to the document ; <i>~theme HorizonDark</i> | ~bg ; Sets the background color of the document ; <i>~bg green</i> | ~fg ; Sets the default text color ; <i>~fg yellow</i> | ~img ; Sets an image as the document background ; <i>~img batman.png</i> | ~box ; Sets a padding specified in pixels to the document ; <i>~box 100</i> | ~box-style ; Sets a visible border to the padding ; <i>~box-style solid</i> | ~set ; Declares a variable and sets a value to it ; <i>~set name Bruce Wayne</i>

$nl 2

!align left
!color green

$text 2.3 The Content Property Keywords

$nl

!color fg

$text The content property keywords start with a <i>!</i>. These properties apply to the entire content that come after the declaration of the content property. These properties define the look of the content in the document.

$nl

$text Here's a table of the the content properties and their uses:

$nl

!align center
!color brown

$table <b>Content Property</b> ; <b>Use</b> ; <b>Example</b> | !bg ;  Sets background for the content ; <i>!bg red</i> | !tab ; Changes the number of spaces in a tab; <i>!tab 8</i> | !font ; Changes the font ; <i>!font Consolas</i> | !size ; Changes the font size ; <i>!size 50</i> | !align ; Chnages the alignment of the content ; <i>!align right</i> | !color ; Changes the foreground of the text ; <i>!color blue</i> | !points-type ; Sets the points to ordered (ol) or unordered (ul); <i>!points-type ol</i> | !points-style ; Sets a style to the points ; <i>!points-style lower-roman | !box ; Sets a padding specified in pixels around the content ; <i>!box 15</i> | !box-style ; Sets a visible border to the padding around the content ; <i>!box-style dotted</i> | !sep ; Sets the delimiter ; <i>!sep ,</i> | !colsep ; Sets the column delimiter ; <i>!colsep -</i> | !dim ; Sets the dimentions of a picture in form: width(px)xheight(px) ; <i>!dim 200x100</i> | !default ; Sets all the content properties to default ; <i>!default</i>

$nl 2

!align left
!color green

$text 2.4 The Content Keywords

$nl

!color fg

$nl

$text The content keywords start with a <i>$</i>. These keywords define the contents of the docuement. Contents include text, pictures, tables, etc.

$nl

$text Here are the content keyword explained:

$nl 2

$text 3. More Utilities
$nl 3

!size 20
!color green
$text 3.1 Themes

!color fg
$nl 2
$text Wave allows documents to have themes. The colors change their shades according to the selected theme. Set the required theme using the <i>~theme</i> page property. Like,

!box 50
!bg %code_bg
!color %code_fg
!font Consolas
$text ~theme HorizonDark
!font Arial
!box 0
!bg bg

!color fg
$text Make sure to add the <i>~theme</i> content property at the very top of the document for the theme to properly take effect.
$nl 2

$text You can choose from the following themes:
$nl
$points Default ; OneDark ; Dracula ; Monokai ; GruvboxLight ; GruvboxDark ; SnazzyLight ; LaserWave ; HorizonLight ; HorizonDark ; Nord ; Synthwave84
$nl
$text Theme names are case sensitive.

$nl 3
!color green
$text 3.2 Variables
!color fg

$nl 2

$text Set variables in a Wave script using the <i>~set</i> page property. Variable names cannot have spaces in them.

!box 50
!bg %code_bg
!color %code_fg
!font Consolas
$text ~set person1 Oliver Queen<br>~set person2 Bruce Wayne
!font Arial
!box 0
!bg bg

!color fg
$text This will set the values of <i>person1</i> and <i>person2</i> to <i>Oliver Queen</i> and <i>Bruce Wayne</i> respectively.
$nl 2

$text Call the variables using the <i>%-</i> symbol. Like,

!box 50
!bg %code_bg
!color %code_fg
!font Consolas
$text ~set person1 Oliver Queen<br>~set person2 Bruce Wayne<br>$text %-person1 meets %-person2.
!font Arial
!box 0
!bg bg

!color fg

$text The value of the same variable can be changed multiple times.

!box 50
!bg %code_bg
!color %code_fg
!font Consolas
$text ~set Robin Jason Todd<br>~set Robin Tim Drake<br>$text %-Robin is the new Robin.
!font Arial
!box 0
!bg bg

!color fg

$text Use <i>%--</i> to escape the <i>%-</i> symbol. Like,

!box 50
!bg %code_bg
!color %code_fg
!font Consolas
$text ~set name Barry Allen<br>$text %--name
!font Arial
!box 0
!bg bg

!color fg

$text This will render <i>%-name</i> in the document and not <i>Barry Allen</i>.

$text 3.3 Special Characters
$nl 2
!color fg
$text Wave also supports special HTML character. Some of the mostly used special characters come as variables in Wave.
$nl
$text Here's the table of some of the special characters available as variables in Wave:
$nl 2

!color brown
!align center
!sep .
$table <b>Variable</b> . <b>Special Character / Value</b> | %-< . %< | %-> . %> | %-& . %& | %-' . %' | %-" . %" | %-- . %- | %-sp . A space character | %-source_name . Name of the Wave file | %-file_name . Name of the transpiled HTML document
!sep ;
!align left
!color fg

$nl 2
$text These special character variables can be overwritten if you use the <i>~set</i> content property on them. Therefore, be careful.
