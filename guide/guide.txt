$text A Wave script is written in a plain text file. Wave scripts do not folow any extensions. You can have an extention for a Wave script or not as long as it's a text file. It's better to have a <i>.txt</i> extension.

$nl

$text A Wave script consists of three different types of keyword:

!color green

$text 2.1 The Page Property Keywords

!color white

$nl
$text The page property keywords start with a <i>~</i>. These properties apply to the entire document. These properties include some important elements that define the look of the document.

$nl

$text Here's a table of the page properties and their uses:

$nl

!align center
!color brown

$table Page Property ; Use ; Example | ~title ; Sets the title to tab in Browser Window ; <i>~title Hello!</i> | ~theme ; Sets theme to the document ; <i>~theme HorizonDark</i> | ~bg ; Sets the background color of the document ; <i>~bg green</i> | ~fg ; Sets the default text color ; <i>~fg yellow</i> | ~img ; Sets an image as the document background ; <i>~img batman.png</i> | ~box ; Sets a padding specified in pixels to the document ; <i>~box 100</i> | ~box-style ; Sets a visible border to the padding ; <i>~box-style solid</i> | ~set ; Declares a variable and sets a value to it ; <i>~set name Bruce Wayne</i>

$nl 2

!align left
!color green

$text 2.2 The Content Property Keywords

$nl

!color white

$text The content property keywords start with a <i>!</i>. These properties apply to the entire content that come after the declaration of the content property. These properties define the look of the content in the document.

$nl

$text Here's a table of the the content properties and their uses:

$nl

!align center
!color brown

$table Content Property ; Use ; Example | !bg ;  Sets background for the content ; <i>!bg red</i> | !tab ; Changes the number of spaces in a tab; <i>!tab 8</i> | !font ; Changes the font ; <i>!font Consolas</i> | !size ; Changes the font size ; <i>!size 50</i> | !align ; Chnages the alignment of the content ; <i>!align right</i> | !color ; Changes the foreground of the text ; <i>!color blue</i> | !points-type ; Sets the points to ordered (ol) or unordered (ul); <i>!points-type ol</i> | !points-style ; Sets a style to the points ; <i>!points-style lower-roman | !box ; Sets a padding specified in pixels around the content ; <i>!box 15</i> | !box-style ; Sets a visible border to the padding around the content ; <i>!box-style dotted</i> | !sep ; Sets the delimiter ; <i>!sep ,</i> | !colsep ; Sets the column delimiter ; <i>!colsep -</i> | !dim ; Sets the dimentions of a picture in form: width(px)xheight(px) ; <i>!dim 200x100</i> | !default ; Sets all the content properties to default ; <i>!default</i>
